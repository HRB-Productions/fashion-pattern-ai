## Retângulo base ABCD
- A = ponto de origem (0, 0)
- B = (busto/4 + ease, 0)                    [ease: malha=0, plano=2]
- C = (busto/4 + ease, comprimento_costas)
- D = (0, comprimento_costas)

## Frente — Ombro
- A→E = (largura_costas/2 + 1)
- E→J = (alt_cava/2) — ponto do ombro

## Frente — Decote
- Decote redondo: raio = busto/20 + 1.5, centro em G
- Decote V: linha diagonal de G até ponto central do busto
- Decote quadrado: recuo de (busto/20) em horizontal e vertical

## Frente — Cava
- L→M = 1.5 cm (ponto de cava superior)
- F→N = (alt_cava - L) / 3
- Curva de cava: spline passando por M → N → J

## Frente — Lateral e Cintura
- A→O = comprimento_blusa (ou comprimento_costas)
- O→P = (cintura/4 + ease + 1)
- Linha lateral: D → P, levemente curva para o caimento

## Ponto do busto
- V = altura_busto a partir do decote
- X = busto/4 - separação_busto (padrão: 9cm para M)

## Top / Malha (sem manga, sem gola, sem pence)
- Usar molde base frente e costas
- Sem curva de manga — fechar a cava
- Comprimento: até a cintura ou conforme especificado
- Pences: não incluir
- Indicar "Cortar 2x (par espelhado)" para frente
- Indicar "Cortar 2x no dobro" para costas

## Camiseta (com manga curta ou longa)
- Usar molde base + módulo de manga
- Manga curta: comprimento = 20cm, largura = alt_cava × 2 - 2
- Manga longa: comprimento = 58cm (M), afunilar para punho
- Casinha da manga: curva que encaixa na cava

## Blusa / Camisa (tecido plano)
- Aplicar ease de tecido plano
- Adicionar margem de botão: 1.5cm centro-frente
- Pences de busto: sim, se possui_pence = true
- Gola: incluir peça separada se possui_gola = true

## Calça
- Frente: gancho frente = quadril/4 - 1
- Costas: gancho costas = quadril/4 + 1.5
- Perna: largura = quadril/4 no topo, afinando para tornozelo

## Short / Bermuda
- Mesma base da calça
- Comprimento: short = 30cm, bermuda = 50cm (a partir da cintura)

## Saia
- Base: retângulo ou A-line
- A-line: abertura = quadril/4 + (comprimento × 0.05)
